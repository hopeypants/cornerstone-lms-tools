üîÑ Updating the Extension

üíæ Export Your Settings Before Updating

Before updating, it's recommended to export your current settings as a backup. Open the extension popup, go to the Global tab, and click "Export Settings" to download a JSON file with all your preferences. This ensures you can restore your settings if needed.

How to Update to the Latest Version

When a new version of the extension is available, follow these steps to update:

  1. Download the latest version:
    Option A: Go to GitHub Repository, click "Code" ‚Üí "Download ZIP", and extract it to the same folder as previously installed (overwriting the old files).
    Option B: If you cloned the repository, pull the latest changes:
    cd cornerstone-lms-tools
    git pull origin main
  2. Open your browser's extension management page:
    • Chrome: Go to chrome://extensions/
    • Edge: Go to edge://extensions/
    • Opera: Go to opera://extensions/
    • Brave/Vivaldi: Similar to Chrome
  3. Update the extension:
    1. Make sure "Developer mode" is enabled (toggle in top right)
    2. Find "Cornerstone LMS Admin Tools" in your extensions list
    3. Click the "Reload" button (circular arrow icon) on the extension card
  4. Verify the update:
    Check that the extension version number matches the latest release on GitHub. You can see the version in the extension popup footer or in the extensions management page.
‚úÖ Settings Are Preserved

Your settings are stored in Chrome's sync storage, so they will be preserved when you update the extension. However, exporting your settings provides an extra safety net in case anything goes wrong.
